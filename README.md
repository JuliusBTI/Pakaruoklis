### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone 
    ```
2. Install 
    ```sh
npm create vite@latest
    ```
3. Install NPM packages
    ```sh
    npm i
    ```
    or
    ```sh
    npm install
    ```
4. Run the server
    ```sh
    npm run dev
    ```
